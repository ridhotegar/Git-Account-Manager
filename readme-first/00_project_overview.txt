PROJECT NAME

Git Account Manager

OBJECTIVE

Build a modern cross-platform desktop application using Rust + Tauri + React + TypeScript.

Target platforms:

- Windows
- Linux
- macOS

The application is designed to manage multiple Git identities from a single desktop application.

The application is NOT another Git client.

It is an account manager for Git.

Primary goals:

- Manage multiple Git accounts
- Switch active account
- Manage SSH keys
- Manage HTTPS credentials
- Support Git CLI
- Support GitHub OAuth
- Support GitLab in future
- Support Bitbucket in future

Non Goals

- Do not build a Git GUI like SourceTree.
- Do not implement commit history visualization.
- Do not replace GitHub Desktop.
- Do not build another IDE.

The application must feel like:

- Docker Desktop
- Postman
- GitHub Desktop

Simple.

Fast.

Native feeling.

Minimal resource usage.
